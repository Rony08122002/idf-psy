# חיילתAI - אפליקציית תמיכה נפשית לחיילים

אפליקציה המספקת תמיכה נפשית ראשונית, אבחון, ומעקב רגשי לחיילים בשירות סדיר, קבע ומילואים.

## מטרת הפרויקט

לספק מענה נפשי נגיש ופרטי לחיילים שזקוקים לתמיכה:
- שיחות יומיות עם דמות וירטואלית (חיילתAI)
- הערכת מצב נפשי ראשונית
- מעקב רגשי לאורך זמן
- הפנייה לגורמי מקצוע (קב"ן) במקרה הצורך

## יכולות עיקריות

- **שיחה עם בינה מלאכותית** - אפשרות לנהל שיחה טקסטואלית עם דמות וירטואלית
- **הערכת מצב נפשי** - שאלון יומי לבדיקת מצב הרגשי של החייל
- **מעקב לאורך זמן** - ניתוח מגמות והשוואת שינויים במצב הנפשי
- **שמירת היסטוריה** - שמירת השיחות והמידע באופן פרטי ומאובטח

## טכנולוגיות

- **React** - ספריית פיתוח ממשק המשתמש
- **TailwindCSS** - מערכת עיצוב
- **Framer Motion** - אנימציות
- **AI API** - חיבור לבינה מלאכותית לניהול שיחות טבעיות

## התקנה והרצה

### דרישות מקדימות
- Node.js גרסה 14 ומעלה
- npm או yarn

### הוראות התקנה

1. **שיבוט (Clone) המאגר:**
```bash
git clone https://github.com/your-username/mental-support-app.git
cd mental-support-app
```

2. **התקנת תלויות:**
```bash
npm install
```

3. **הגדרת משתני סביבה:**
צור קובץ `.env` בתיקיית הפרויקט עם הערך הבא:
```
REACT_APP_OPENAI_API_KEY=your_api_key_here
```

4. **הרצת האפליקציה בסביבת פיתוח:**
```bash
npm run dev
```

האפליקציה תרוץ ב-`http://localhost:5173`
